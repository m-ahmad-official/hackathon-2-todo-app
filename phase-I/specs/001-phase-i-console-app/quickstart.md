# Quickstart: Phase I - In-Memory Console Todo App

## Prerequisites
- Python 3.11+

## Execution
To run the Phase I application:
```bash
python main.py
```

## Usage
The application will present a menu:
1. Add Task
2. View Task List
3. Update Task
4. Delete Task
5. Mark Task Complete
6. Mark Task Incomplete
7. Exit

Follow the on-screen prompts to interact with your tasks. Note that all data is lost when you exit (Option 7 or terminating the process).
